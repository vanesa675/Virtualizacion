Ventajas

  • Ahorro de costes:
    1. - Reduce la necesidad de comprar hardware físico.
    2. - Reduce el consumo de energía.
    3. - Reduce la necesidad de espacio físico para servidores.
    4. - Reduce la necesidad de personal para la administración de los equipos.

  • Mayor eficiencia: Permite aprovechar al máximo la capacidad del hardware. Simplifica la gestión de la infraestructura informática. Permite realizar pruebas y actualizaciones sin afectar al sistema en producción.

  • Flexibilidad: Permite crear y eliminar VM fácilmente. Permite adaptar la infraestructura informática a las necesidades cambiantes del negocio. Permite crear VM para diferentes proyectos, departamentos o usuarios.

  • Escalabilidad: Permite aumentar o disminuir la capacidad de las VM según las necesidades. Permite agregar más recursos a una VM existente o crear nuevas VM para satisfacer la demanda.

  • Seguridad: Permite aislar diferentes sistemas operativos y aplicaciones en diferentes VM. Permite crear VM para diferentes niveles de seguridad. Permite separar aplicaciones sensibles del resto del sistema.

  • Mayor disponibilidad: Permite realizar pruebas y actualizaciones sin afectar al sistema en producción. Permite crear VM de alta disponibilidad para garantizar la continuidad del negocio en caso de un fallo del equipo físico.

  • Facilitar la recuperación ante desastres: Permite crear copias de seguridad de las VM y restaurarlas rápidamente en caso de un desastre.
    Permite minimizar el tiempo de inactividad y los daños causados por un fallo del equipo físico.

  • Mayor eficiencia energética: Permite utilizar los recursos de hardware de manera más eficiente. Reduce el consumo de energía.

  • Mayor portabilidad: Las VM se pueden mover fácilmente entre diferentes equipos físicos. Permite mayor flexibilidad y portabilidad.

  • Facilidad de uso: Existen herramientas que permiten administrar las VM de forma sencilla e intuitiva.
